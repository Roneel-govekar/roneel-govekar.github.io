Fashion Bazar App

Home Screen

This is the home screen of the Fashion Bazar app. The user will open either the left or right door. Left door leads to the wardrobe feature and right door leads to a secret future feature (WIP).

This app also provides way to login using email for better performance for the wardrobe feature.

Wardrobe Feature

The user can add pictures of their clothing items to their virtual wardrobe using the '+' button.

The app uses on-device image categorization to automatically organize and tag clothing items. It also supports online image categorization using GROK api for additional clothing item identification.

When the refresh button is pressed, the app provides an outfit based on the wardrobe and nearest occasion from the timetable with any new clothing items added by the user.

Calendar Feature

The user can their weekly timetable along with the occasion and the time of the event.

If the user has a meeting on Friday at 3 PM, they can set an event in the calendar and the app will recommend appropriate clothing for that occasion and taking weather into account.
